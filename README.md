# Single_page-form
 This is a code block that generates a simple feedback form. The form has fields for name, email, comment, and some options to choose from. The options include checkboxes, radio buttons, and a dropdown list of cars. When the user submits the form, all the details are displayed on the console and in an alert box.

# Features
- This form allows you to enter your personal information and preferences on a single page.
- You can fill in your name, email and comment in the text fields provided.
- You can also select from various options using the checkbox, radio button and dropdown list components.
- The radio button lets you choose one item from a list of options, such as online or offline.
- The dropdown list lets you choose one item from a list of cars, such as Audi, BMW or Fortuner.
- After you submit the form, all the details you entered will be displayed on the console and also in an alert box.
- This way, you can review your input and confirm that it is correct and complete.
